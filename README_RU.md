# FLSampler (Foveated Latent Sampler) (BSS)

**FLSampler (BSS)** — это высокотехнологичный пользовательский сэмплер для ComfyUI, вдохновленный принципами физиологии человеческого зрения (**Foveated Vision** / Фовеальное зрение). Он предназначен для получения кинематографичной четкости, прорисовки богатых микро-текстур и управления глубиной детализации в латентном пространстве без увеличения времени генерации.

## 🔬 Физика и математика процесса

Стандартные сэмплеры (KSampler) применяют шум и очистку равномерно ко всему скрытому пространству (latent space), что часто приводит либо к замыливанию мелких текстур, либо к перегрузке изображения шумом и артефактами (пережженности).

**FLSampler** решает эту проблему за счет интеллектуального избирательного воздействия:

1. **Дифференциальный анализ латента (Focus Detection):**
   На каждом шаге очистки шума сэмплер вычисляет абсолютную разницу $\Delta = |x_{0, t} - x_{0, t-1}|$ между текущим предсказанным чистым изображением $x_0$ и его состоянием на предыдущем шаге. Это позволяет с высокой точностью локализовать области, где в данный момент формируется геометрия (контуры лица, границы объектов, мелкие детали).
2. **Адаптивная карта Fovea с инерцией (Momentum Mask):**
   Разностная карта сглаживается с помощью пулинга (`avg_pool2d`), после чего для нее рассчитывается динамический порог на основе среднего значения и стандартного отклонения: 
   $$Threshold = \mu + 0.5 \cdot \sigma$$
   Через сигмоидную активацию формируется мягкая маска фокуса зрения (**Fovea Mask**). Для предотвращения резких скачков маски между шагами применяется инерция (**Momentum** / `mask_inertia`), обеспечивающая плавное перетекание зон детализации.
3. **Локальный контраст (Sharpness Boost):**
   В зонах высокой активности Fovea применяется метод **Unsharp Masking** непосредственно в скрытом пространстве. Из оригинального тензора вычитается его размытая версия для получения карты высоких частот (High Frequencies), которая затем точечно усиливается. Это придает границам объектов невероятную четкость и кинематографичную глубину.
4. **Стохастическое инжектирование текстуры (Texture Grain):**
   В активные области фокуса на ранних и средних этапах генерации подмешивается контролируемый микро-шум, который на последующих шагах сэмплинга превращается в реалистичные высокочастотные текстуры (поры кожи, ткань одежды, текстура дерева, листья, шерсть), избегая пластикового "замыливания". На поздних шагах воздействие плавно затухает (`decay`), гарантируя чистоту финального кадра.

## 🎛️ Параметры и их влияние

Нода полностью заменяет стандартный KSampler и содержит как базовые, так и уникальные физические регуляторы:

| Параметр | Диапазон | Рекомендуемое | Описание |
| :--- | :--- | :--- | :--- |
| **fovea_strength** | `0.0 – 10.0` | **`3.0 – 8.0`** | **Сила текстуры:** отвечает за микро-детализацию, пористость и проработку мелких текстур. При `0.0` инжекция шума отключена. По умолчанию: `3.0`. |
| **sharpness** | `0.0 – 3.0` | **`0.3 – 1.0`** | **Локальный контраст:** отвечает за очерченность границ и резкость контуров объектов. Предотвращает пережженность кадра в зонах фокуса. По умолчанию: `0.5`. |
| **mask_inertia** | `0.0 – 0.99` | **`0.80 – 0.90`** | **Инерция маски:** степень плавности перехода фокуса между шагами. Высокие значения (`>0.85`) делают детали более стабильными, низкие заставляют фокус реагировать мгновенно. По умолчанию: `0.85` (шаг `0.01`). |
| **denoise** | `0.0 – 1.0` | **`1.0`** | Сила денойзинга (классический параметр). |
| **steps** / **cfg** | - | - | Количество шагов и сила соответствия промпту. |

## 📊 Выходы ноды

Нода имеет два выхода:
1. 🖼️ **latent**: Оптимизированный латентный тензор для последующего декодирования через VAE Decode.
2. 👁️ **fovea_mask**: Визуализированная в формате RGB (IMAGE) накопленная карта фокуса Fovea. Вы можете передать её в Save Image или Preview Image, чтобы увидеть, на каких именно областях концентрировался сэмплер в процессе генерации. Это беспрецедентный инструмент для отладки воркфлоу!

## 📋 Рекомендуемый воркфлоу (Workflow)

```
[ MODEL (например, Anima / SDXL) ] ──> [ positive ]
                                 ──> [ negative ]
                                          ↓
[ Latent Image (разрешение) ] ──────> [ latent_image ] ──> [ FLSampler (BSS) ] ──(latent)──> [ VAE Decode ] ──> [ Save Image ]
                                                                   └──(fovea_mask)──> [ Preview Image ]
```
